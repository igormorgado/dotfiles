		  DarkTrial color scheme    by Igor

DarkTrial, a color scheme with a unique touch.

==============================================================================
Color scheme options					darktrial-options*

						*g:darktrial_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:darktrial_transp_bg = 0
<
